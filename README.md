# 中国传统色数据

## 数据来源

本数据按照《中国传统色》一书中提供的 RGB 数值进行摘录。数据主要供自己做图配色使用，感兴趣购买原书。

## RoadMap

该仓库未来计划实现的功能：

- [ ]: 输入颜色，返回最接近的颜色和名字
- [ ]: 用已有数据生成漂亮的颜色卡片
- [ ]: 使用最接近的颜色替换一张图片
